# Automated Whitelist Merger

Инструмент для автоматического сбора и объединения белых списков (whitelists) из внешних источников в один финальный файл.

## 📌 Описание проекта
Скрипт автоматически отслеживает изменения, скачивает актуальные белые списки из удаленного репозитория и консолидирует их в единый перечень. Это полностью исключает ручную работу и гарантирует актуальность данных в реальном времени.

## 🔍 Источник данных
Все исходные белые списки автоматически загружаются из этого репозитория:
👉 [https://github.com](https://github.com/igareck/vpn-configs-for-russia)

## 🚀 Основные функции
* **Полная автоматизация**: сбор данных по расписанию без участия пользователя.
* **Валидация**: строгая проверка строк на корректность форматов.

## ⚙️ Инструкция по использованию в VPN-клиенте

Вам достаточно один раз добавить ссылку в ваше приложение, и список будет обновляться сам.

1. **Скопируйте прямую ссылку** на ваш итоговый объединенный файл.
2. **Откройте настройки** вашего VPN-клиента (например, Nekobox, Amnezia, Shadowrocket, Sing-box).
3. **Вставьте скопированную ссылку** в поле для URL-адреса списка.
4. **Включите автообновление** в настройках клиента, чтобы всегда иметь актуальные данные.
